NLP Overview: Natural Language Processing is a concept used in ML for text preprocessing. Machines dont understand plain english or any other language, so nlp helps by offereing techniques like tokenization, stemming, lemmatization, stopwords, Bag of Words, unigram etc.
All these help in text preprocessing, and tyhe more we use these techniques for model building, training and testing, then the more accurate model will be.

Leveraged multiple libraries from nltk, such as:
### tokenization (sent_tokenize, word_tokenize)
### stemming (PorterStem, RegexPStem, SnowballStem)
### Lemmatization
### Stopwords
### Named Entity Recognition
